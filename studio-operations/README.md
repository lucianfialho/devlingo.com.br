## Emergency Procedures

### Site Down
1. Check Vercel status
2. Check Redis connection
3. Check Firebase quota
4. Alert: Engineering Lead
5. Fallback: Static cached version

### Traffic Drop (>20%)
1. Check Google Search Console
2. Check for manual actions
3. Check indexing status
4. Alert: SEO Lead
5. Investigate algorithm update

### Revenue Drop (>20%)
1. Check AdSense account status
2. Check ad placements rendering
3. Check traffic quality
4. Alert: Operations Manager
5. Review CPM by category

### Security Issue
1. Immediately notify Engineering Lead
2. Check Vercel logs
3. Check Firebase auth
4. Rotate API keys if needed
